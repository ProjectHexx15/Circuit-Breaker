<h1>Circuit Breaker</h1>

Circuit Breaker is a 3D FPS set in an evil factory that you need to try and escape! It features smoooth, standard FPS controls and has unique 3D pixel art graphics that makes the game stand out. It features multiple doors that you need to unlock by using keys stolen from enemy guards. This game was created for my coursework in college.

- Fun and natural controls
- Simple enenemy guard robots
- health and ammo recharge stations
- Unique artstyle
